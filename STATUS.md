# OpenClash 规则重写状态

## 当前状态

已建立本地重写工作台，并已初始化为独立 Git 仓库。初始提交已完成。

当前处于逐组确认阶段，正在等待确认第 1 组“基础策略组与通用选择”。

## 已确认事实

- 当前模板没有链式代理模块。
- 当前模板缺少通用总控组。
- 当前模板的业务组默认继承 `default`，而 `default` 第一项是 `直连`。
- 当前模板没有通用广告拒绝规则。
- liandu2024 的 `Block.list` 只包含 Adobe 相关域名，不是广告规则。
- liandu2024 的自定义规则存在个人偏好，应按组拆开审查。
- **不依赖 liandu2024 的外部仓库，所有规则在本项目内自行维护创建。**
- 用户确认：`03-项目/` 下的具体项目可以使用独立 Git 仓库进行版本控制，不等同于 OrbitOS 根仓库边界。

## 已确认

- ✅ 第 1 组：基础策略组与通用选择（2026-06-17）
- ✅ 第 2 组：国内、私有网络、自用直连 + 回家场景适配（2026-06-17）
- ✅ 第 3 组：广告/拦截规则（2026-06-17）

## 待确认
